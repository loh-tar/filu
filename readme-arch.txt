Arch Specific Notes
=====================
To compile the programs you need:
  base-devel
  cmake
  qt4
  muparser
  ta-lib          Available in AUR

The provider scripts needs the packages:
  perl-io-html
  perl-libwww
  perl-timedate

Some more packages available in AUR:
  perl-cache-cache
  perl-date-simple
  perl-xml-libxml-simple
  perl-html-tableextract

The database:
  postgresql
